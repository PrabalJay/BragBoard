# BragBoard: Internal Employee Recognition Wall 🚀

**BragBoard** is a full-stack social recognition platform designed to boost workplace morale and gamify employee appreciation. This project was developed over 8 weeks during my Virtual Internship at **Infosys Springboard**.

## 🌟 Key Features
* **Secure Access:** JWT-based authentication with role-based permissions (Admin vs. Employee).
* **Social Feed:** A centralized, real-time wall supporting text and Rich Media (Image/Video) uploads.
* **Interactive Engagement:** Social reactions (Like, Clap, Star) and threaded comment sections for peer-to-peer feedback.
* **Admin Suite:** Comprehensive analytics dashboard displaying "Top Contributors", content moderation tools (Reporting/Deletion), and CSV data exports.
* **Notifications:** Real-time alerts when you are tagged or recognized by a teammate.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS, Lucide Icons
* **Backend:** FastAPI (Python), SQLAlchemy ORM
* **Database:** SQLite (`bragboard.db`)
* **Deployment:** Netlify (Frontend) & Render (Backend)

## 📈 8-Week Development Journey
1. **Weeks 1-2:** Foundation, UI setup, and secure JWT Authentication.
2. **Weeks 3-4:** Shout-out creation, recipient tagging, and rich media integration.
3. **Weeks 5-6:** The Social Loop (Toggle reactions, real-time counters, and threaded comments).
4. **Weeks 7-8:** Admin Governance, leaderboard analytics, reporting mechanisms, and CSV export functionality.
